# FitTrack+

Personalized fitness & nutrition dashboard built with Next.js and Tailwind CSS.

This version includes:
- Branded login page (`/login`) with demo auth
- Sample workouts, meals, and goals preloaded on first login
- Enhanced dashboard with Recharts (weekly calories, weekly workouts, macro donut)

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser.
4. You will be redirected to `/login`. Enter any email/password to continue.
